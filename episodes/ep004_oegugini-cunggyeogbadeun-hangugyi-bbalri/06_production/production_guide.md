## 2. 편집 가이드라인

### 전체 톤 & 스타일
- **스타일**: 정보성 브이로그 + 살짝 유머 + 진정성
- **페이스**: 인트로는 빠르게(훅 잡기), 본론 4에서 살짝 느리게(감성 터치), 아웃트로 다시 밝게
- **언어**: 한국어 구어체. 진행자 외국인 억양 그대로 살리는 게 오히려 매력 포인트

### 컷 편집 리듬
| 구간 | 컷 길이 | 비고 |
|------|---------|------|
| 인트로 (0:00~0:50) | 3~5초 | 빠른 훅, 배달 도착 서프라이즈 |
| 본론 1~2 (0:50~4:00) | 3~4초 | 정보 전달 + 리액션 컷 병행 |
| 본론 3 (4:00~5:30) | 4~5초 | B-roll 중심, 여유있게 |
| 본론 4 (5:30~7:00) | 5~6초 | 진지한 톤, 긴 컷 |
| 아웃트로 (7:00~8:00) | 3~4초 | 타임랩스 + 마무리 |

### 자막 스타일
- **본문 자막**: 나눔스퀘어 Bold, 크기 52pt, 흰색, 검정 아웃라인 2px, 하단 1/6 위치
- **강조 자막 (수치/핵심)**: 노란색 또는 민트 (#4ECDC4), 크기 1.4배, 배경 반투명 박스, 화면 중앙
- **섹션 전환 자막**: 반투명 어두운 배경 + 이모지 아이콘 + Bold 텍스트

## 4. 썸네일 컨셉

### 안 A — 배달 속도 강조
- **메인 이미지**: 외국인이 28분 배달 예상 시간 보며 입 벌리고 놀란 표정 (스마트폰 화면 함께)
- **텍스트**: `"28분?! 🍕⚡"` (빨간색 대형 폰트) + 하단 `한국 빨리빨리 문화의 진실`
- **배경**: 서울 야경 + 배달 모티프

### 안 B — 속도 비교 강조
- **메인 이미지**: 외국인 vs 한국인 나란히 — 외국인은 시계 들고 기다리는 표정, 한국인은 이미 다 먹은 표정
- **텍스트**: `"왜 한국인은 이렇게 빠를까?"` (흰색 폰트, 검정 배경)
- **배경**: 분할 화면 (회색 느린 느낌 vs 한국 빨간/빠른 느낌)

> **추천**: 안 A — 숫자(28분)가 직접적인 클릭 유발 요소로 더 효과적
